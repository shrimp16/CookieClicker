# CookieClicker

Simple game where you have to click on the big cookie in order to gain more cookies and buy new strucutres and build a cookie empire.

# Technologies:
  - HTML
  - CSS
  - JavaScript
  - NPM
  - NodeJS
  - Express.js
  - FS.js
